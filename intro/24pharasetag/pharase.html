HTML

  • Abbreviation: < abbr title = "" >

    • abbreviation tag defines an abbreviation or an acronym, like "HTML", "CSS", "Mr.", "Dr.", "ASAP", "ATM".

  • Marked: < mark >

    • mark tag defines text that should be marked or highlighted.

  • Strong: < strong >

    • strong tag is used to define text with strong importance.

  • Emphasized: < em >

    • emphasized tag is used to define emphasized text. The content inside is typically displayed in italic

  • Definition: < dfn >

    • tag stands for the "definition element", and it specifies a term that is going to be defined within the content.

  • Quoting: < blockquote cite="" >,

    • for Short Quotations

  • Short: < q >

    • tag defines smaller text

  • Code: < code >

    • tag is used to define a piece of computer code

  • Keyboard: < kbd >

    • tag is used to define keyboard input.

  • Address: < address >
