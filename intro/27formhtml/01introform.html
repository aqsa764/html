there are two type of method

  1. GET
  2. POST

GET:-

  • GET will be sending form data using the URL
  • GET will append data to url using ? = query string and key=value&key=value
  • GET method is less secure because its shows form data in url
  • GET method have limitation of 2048 char only 1024 bites
  • GET method doesnt support file uploading
  • GET method help to get data back after refreshing or reloading
  • GET method data will be bookmarked
  • GET method is fater than post
  • GET method preferrable while searching

FOR EXAMPLE:file:///C:/Users/tops/Desktop/intro/01introform.html?username=aqsa&password=aqaswdw&btn=Submit

post:-

  • POST will be sending form data without using the URL
  • POST method is secure because its not sending data by URL
  • There is no limitation for POST method for seding data to the server
  • POST method support file uploading
  • while file uploading we must required to add enctype="multipart/form-data" attr
  • we are not getting data after refreshing/realoading our page in POST method
  • wnt able to do bookmarked post method form data
  • POST method is slower than get
  • POST method preferrable while securely sending data
